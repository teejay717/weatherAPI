## üöÄ Setup & Installation

To run this project locally, you will need to obtain your own API key from OpenWeatherMap.

1.  **Clone the repository:**
    ```bash
    git clone [Your Repository URL]
    cd weatherAPI
    ```
2.  **Get an API Key:**
    * Sign up for a free account at [OpenWeatherMap](https://openweathermap.org/api).
    * Generate a new API key from your account dashboard.
3.  **Update the Code:**
    * Open the `app.js` file.
    * Locate the section where the API key is used (or will be used).
    * **Replace the placeholder/empty string** with your actual OpenWeatherMap API key.

    *Note: The API key has been removed from the public repository for security, as seen in the commit history.*

4.  **Launch the Application:**
    * Simply open the `index.html` file in your preferred web browser.
